# Iris Flower Prediction App 🌸

A Streamlit web application that predicts **Iris flower species** using a Machine Learning model.

## Features

* Input flower measurements
* Predict the Iris species
* Simple interactive interface built with Streamlit

## How to Run Locally

```bash
pip install -r requirements.txt
streamlit run iris_app.py
```
